445/**
446 * @ingroup Socket_register_group
447 * @brief Socket command register(R/W)
448 * @details This is used to set the command for Socket n such as OPEN, CLOSE, CONNECT, LISTEN, SEND, and RECEIVE.\n
449 * After W5500 accepts the command, the @ref Sn_CR register is automatically cleared to 0x00.
450 * Even though @ref Sn_CR is cleared to 0x00, the command is still being processed.\n
451 * To check whether the command is completed or not, please check the @ref Sn_IR or @ref Sn_SR.
452 * - @ref Sn_CR_OPEN : Initialize or open socket.
453 * - @ref Sn_CR_LISTEN : Wait connection request in TCP mode(<b>Server mode</b>)
454 * - @ref Sn_CR_CONNECT : Send connection request in TCP mode(<b>Client mode</b>)
455 * - @ref Sn_CR_DISCON : Send closing request in TCP mode.
